## 문제 설명

길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요.
예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

## 제한 조건

- n은 길이 10,000이하인 자연수입니다.

## 입출력 예

| n   | return     |
| --- | ---------- |
| 3   | "수박수"   |
| 4   | "수박수박" |

## 작성 답안

```js
function solution(n) {
  return Array(n)
    .fill()
    .map((e, i) => (i % 2 ? "박" : "수"))
    .join("");
}
```

기존 문자열을 `split`으로 배열로 바꾸거나 `Array`로 만든 배열을 이용하는 방법을 반복하니 점점 익숙해지는 것 같아요.
데이비드 말란 교수님의 CS강의를 들은 적 있는데, **문자열은 문자로 이루어진 배열**이라는 말씀이 기억에 많이 남네요
