## 문제 설명

정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.

## 제한 조건

- n은 0 이상 3000이하인 정수입니다.

## 입출력 예

| n   | return |
| --- | ------ |
| 12  | 28     |
| 5   | 6      |

## 작성 답안

```js
function solution(n) {
  let answer = 0;
  for (let i = 1; i <= n / 2; i++) {
    if (!(n % i)) {
      answer += i;
    }
  }
  return answer + n;
}
```

간단하게 모든 약수를 더해주는 방식을 구현했습니다. 한 줄로는 어떻게 해야할지 모르겠더라구요.
n/2까지만 반복문을 도는 이유는 n/2 이후에 약수는 자기자신뿐이기 때문입니다.
예를 들어 12의 약수는 모두 1, 2, 3, 4, 6, 12 인데 12의 절반인 6 이후로 약수는 자기자신만 존재하고 있습니다.
따라서 n/2까지만 반복한 후, 마지막에 자기자신인 n을 더해주었습니다.

처음에는 n\*\*0.5 즉, 제곱근까지만 반복하는 식으로 했었는데, 12의 경우 제곱근이 3과 4 사이여서 3과 짝을 이루는 4를 찾아 더해주는 내용이 필요해지더라구요.
마찬가지로 1과 짝을 이루는 12, 2와 짝을 이루는 6까지도 찾아서 더해주어야 합니다.
두번째 방식으로 코드를 채점해봤을때, 3가지 테스트 케이스에서 실패하는데 한가지는 1의 경우라 금방 해결했지만, 나머지 두가지는 어떤 경우인지 아직 모르겠습니다.
