## 문제 설명

함수 solution은 정수 n을 매개변수로 입력받습니다. n의 각 자릿수를 큰것부터 작은 순으로 정렬한 새로운 정수를 리턴해주세요. 예를들어 n이 118372면 873211을 리턴하면 됩니다.

## 제한 조건

- n은 1이상 8000000000 이하인 자연수입니다.

## 입출력 예

| n      | return |
| ------ | ------ |
| 118372 | 873211 |

## 작성 답안

```js
function solution(n) {
  return (
    `${n}`
      .split("")
      .sort((a, b) => b - a)
      .join("") * 1
  );
}
```

문자열로 바꾼 후 쪼개서 정렬을 했습니다.
이 과정에서 습관적으로 내림차순 하는 정렬 함수를 넣었는데, 다른 분들의 풀이를 보니 정렬 함수를 넣지 않고(오름차순 정렬) 후에 `reverse()`를 이용해 뒤집는 경우를 보았어요.

메소드를 쓰지 않는 다른 방법으로는 반복문이나 재귀로 10으로 나눈 나머지를 문자열에 더해가는 방식이 생각나네요.
