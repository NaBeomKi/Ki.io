## 문제 설명

임의의 양의 정수 n에 대해, n이 어떤 양의 정수 x의 제곱인지 아닌지 판단하려 합니다.
n이 양의 정수 x의 제곱이라면 x+1의 제곱을 리턴하고, n이 양의 정수 x의 제곱이 아니라면 -1을 리턴하는 함수를 완성하세요.

## 제한 조건

- n은 1이상, 50000000000000 이하인 양의 정수입니다.

## 입출력 예

| n   | return |
| --- | ------ |
| 121 | 144    |
| 3   | -1     |

## 작성 답안

```js
function solution(n) {
  return Math.sqrt(n).toString().includes(".") ? -1 : (Math.sqrt(n) + 1) ** 2;
}
```

`Math.sqrt()`함수를 이용했는데, `n**0.5`도 동일하게 제곱근이 나온다는걸 나중에 알았습니다.
그리고 저는 계산된 결과를 문자열로 형변환해 소수점이 포함되어 있는지로 정수인지 아닌지를 판단했습니다.
다른 분들의 경우 `parseInt`를 이용해 정수로 바꾼 후 계산한 제곱근과 동일한지 비교하는 식으로도 하신 경우도 봤습니다.

`parseInt`가 이름처럼 정수형으로 변환한다는 본래의 기능을 잊고 단순히 숫자로 이루어진 문자열을 숫자로 변환할 때만 사용하게 되는 경향이 있는데 반성하게 되더라구요.(그리고 또 잊어먹겠지만...)

메소드 체이닝을 이용해 한 줄로 끝내는 것도 재미있지만, 메소드를 이용하지 않고 간단하게 계산할 수 있는 방법들을 알아가는 것도 재미있네요.
