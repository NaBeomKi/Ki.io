## 문제 설명

프로그래머스 모바일은 개인정보 보호를 위해 고지서를 보낼 때 고객들의 전화번호의 일부를 가립니다.
전화번호가 문자열 phone_number로 주어졌을 때, 전화번호의 뒷 4자리를 제외한 나머지 숫자를 전부 \*으로 가린 문자열을 리턴하는 함수, solution을 완성해주세요.

## 제한 조건

- s는 길이 4 이상, 20이하인 문자열입니다.

## 입출력 예

| phone_number  | return           |
| ------------- | ---------------- |
| "01033334444" | "**\*\*\***4444" |
| "027778888"   | "**\***8888"     |

## 작성 답안

```js
function solution(phone_number) {
  return phone_number
    .split("")
    .map((n, i) => (i < phone_number.length - 4 ? "*" : n))
    .join("");
}
```

정규식을 썼어도 좋았을텐데 자꾸 쓰질 않으니 외우고 잊어먹고의 반복이다.

정규식 이외에 다른 좋은 답안들 중에서, 눈에 들어온 더 간단하고 직관적으로 느껴진 아래의 코드

```js
function solution(s) {
  return "*".repeat(s.length - 4) + s.slice(-4);
}
```

그 밖에 반복문으로 구현하는 방법도 좋았다. 오히려 더 직관적으로 느껴진 면도 있었고,
위에 사용한 함수들이 성능을 얼마나 먹는지까지는 모르겠지만 그래도 메소드보단 그 편이 더 효율성이 높을거라는 생각이 들기도 하고...
