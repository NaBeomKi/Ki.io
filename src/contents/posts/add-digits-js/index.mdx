## 문제 설명

자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.
예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

## 제한 조건

- N의 범위 : 100,000,000 이하의 자연수

## 입출력 예

| N   | answer |
| --- | ------ |
| 123 | 6      |
| 987 | 24     |

## 작성 답안

```js
function solution(n) {
  return `${n}`.split("").reduce((a, b) => a * 1 + b * 1, 0);
}
```

먼저 숫자를 문자열로 바꾼뒤 `split`을 이용해 각 자릿수를 나눠 배열로 만들고, `reduce`를 이용해 배열의 원소들을 더해 문제를 해결했다.
`split`으로 각 자릿수를 나눠 배열로 반환을 할 때, 이 배열에 담긴 숫자들의 자료형은 문자열이다.
때문에 `reduce`로 연산을 할 때 다시 숫자로 형변환을 해주지 않으면 문자열의 더하기 연산이 이루어지기 때문에 형변환을 해주어야 한다.

`reduce`의 callback에서 첫번째 인자는 콜백 함수의 리턴값(누적된 값) 초기 값이 숫자인 상황이라면 사실 형변환을 하지 않아도 문제가 되지 않겠지만,
두번째 인자의 경우 문자열인 숫자를 받기 때문에 반드시 형변환을 해주어야 한다.

마지막으로 하나 더 신경써야하는 점은 0이 입력 값으로 들어왔을때, 0을 반환해야 한다는 것이다.
초기값 0을 넣어주지 않으면 배열의 원소가 1개이기 때문에 바로 해당 원소를 반환해버리는데 앞서 이야기했지만 원소가 문자열이기 때문에 테스트 케이스를 통과할 수가 없다.
